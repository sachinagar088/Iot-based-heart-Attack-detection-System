# Iot-based-Heart-Attack-Detection-System

# Components Required
1. Arduino <br>
2. Heart Beat Sensor<br>
3. Jumper Wires <br>

# Working

With the help of heart beat sensor data is captured.<br>
If any abnormal pattern of Heart rate is noticed the System Alerts the user.
